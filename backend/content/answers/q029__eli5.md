question: "How does linear regression work and what are its key assumptions?"

**Linear regression** is like fitting the single straightest line through a scatter of dots so it best captures the trend. Say you plot house size against price: the line lets you read off a predicted price for any size. The model just picks a slope and starting point that make the line sit as close as possible to all the dots — specifically, minimizing the **squared gaps** between the line and each real point. The nice part is readability: each slope says "every extra square foot adds about $X." But the straight-line trick only trusts a few assumptions: the real relationship is roughly **linear**, the errors are **independent** with **steady spread**, and features aren't near-duplicates of each other. Break those — say the pattern actually curves — and the line misleads, so you'd transform the data or switch to a different model.
