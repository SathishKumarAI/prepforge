Ask the simplest possible modeling question: if I must predict y from features x using the *least* structure, what do I assume? A **linear** relationship — y is a weighted sum of features plus noise. Linear regression finds the weights that make this line/plane fit best.

"Best" needs definition. We minimize the sum of **squared** residuals. Why squared, not absolute? Two deep reasons converge: geometrically, minimizing squared error is an **orthogonal projection** of y onto the column space of X — the residual is perpendicular to the fitted subspace, giving a unique closed-form solution (the normal equations). Probabilistically, if the noise is Gaussian, least-squares *is* maximum likelihood. So the squared loss isn't arbitrary — it's what Gaussian noise and Euclidean geometry both demand.

The assumptions are exactly the conditions under which this is valid and optimal:

- **Linearity** — the true mean is linear in the parameters.
- **Independence** of errors.
- **Homoscedasticity** — constant error variance (else some points are wrongly weighted).
- **Normality** of errors — needed for valid confidence intervals/tests, not for the point estimate.
- **No perfect multicollinearity** — else weights aren't identifiable.

The non-obvious insight: linear regression can model highly *nonlinear* phenomena — "linear" refers to linearity in the *parameters*, so with basis expansions (polynomials, splines) the same machinery fits curves.
