question: "How can you estimate causal effects from observational data when you cannot run an experiment?"
topic: "Data Science"

The fundamental obstacle: causation is about **counterfactuals** — what would have happened to the *same* unit under the other treatment — and you can never observe both. Randomization normally solves this by making treated and untreated groups exchangeable, so the untreated act as a valid stand-in for the treated's counterfactual. Without an experiment, treatment is *chosen*, so it correlates with confounders — variables that drive both who gets treated and the outcome. Naive comparison then conflates the effect with the confounding.

Every observational method is a strategy to *recreate the missing randomization*:

- **Adjustment / matching / propensity scores:** if you can measure all confounders, condition on them so that within strata treatment is "as good as random." Assumption: no unmeasured confounding (ignorability) — untestable and the Achilles' heel.
- **Instrumental variables:** find a variable that shifts treatment but affects the outcome *only through* treatment (like a lottery or policy quirk). It injects exogenous variation — nature's mini-experiment.
- **Difference-in-differences / regression discontinuity:** exploit timing or an arbitrary cutoff so units just above/below are comparable, differencing out fixed confounders.

The unifying principle: you can't get causation from data alone — you must import *assumptions* (encoded in a causal graph) about which paths exist. Data constrains, assumptions identify.

The insight most miss: causal claims live or die on the *credibility of the untestable assumption*, not on model fit — a higher R² never rescues a confounded design.
