question: "What is the difference between L1 and L2 regularization?"
topic: "Machine Learning"
model: "Claude Code (authored)"
sources: []
---

Both **L1** and **L2 regularization** stop a model from leaning too hard on any feature — imagine a strict budget on how large its "importance dials" can grow, which discourages overfitting. The difference is how the budget punishes. **L2 (Ridge)** charges based on the *square* of each dial, so it aggressively shrinks the biggest dials but lets everything keep a little influence — like a tax that gently turns all knobs down toward zero without ever switching them off. **L1 (Lasso)** charges the *plain size* of each dial, and this shape actually pushes weak dials **all the way to exactly zero**, effectively deleting useless features — automatic feature selection and a simpler, sparser model. A knob called **lambda** sets how strict the budget is. And **Elastic Net** simply blends both penalties to get a bit of each behavior.
