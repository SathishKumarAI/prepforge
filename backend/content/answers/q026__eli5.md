question: "What is the difference between classification and regression?"
topic: "Machine Learning"
model: "Claude Code (authored)"
sources: []
---

Both are types of learning from labeled examples, and the only real difference is *what kind of answer* you want. Imagine two vending machines. The first has **buttons**: press one and you get an item from a fixed set — cola, water, juice. That's **classification**: the model sorts each input into a category, like "spam or not spam" or "which digit is this." The second machine is a **fuel pump**: it hands back a number that can slide anywhere along a scale — 3.2 liters, 47.9 dollars. That's **regression**: predicting a continuous amount, like a house price or tomorrow's temperature. Same idea (learn from examples), different output. Watch out for one naming trap: **logistic regression** is actually a *classification* method despite its name, because it predicts which category something belongs to, not a free-floating number.
